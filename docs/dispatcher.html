Dispatcher

The dispatcher is the central hub in which all application data flows throw. When a store is created it registers with the dispatcher a callback with the dispatcher. Whenever an action creators create actions it is passed to the dispatcher which passes it to any registered stores. The dispatcher will call each registered callback synchronously. Any actions that are dispatched during this process will be queued until all callbacks are called.

Marty uses facebook's dispatcher which is a singleton accessible on Marty.

var dispatchToken = Marty.dispatcher.register(function (action) {
  console.log(action);
});

var action = new Action('ADD_FOO', [], Marty.constants.actionSources.VIEW);
Marty.dispatcher.dispatch(action);
