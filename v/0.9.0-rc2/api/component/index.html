Component API

class User extends Marty.Component {
  constructor(props, context) {
    super(props, context);
    this.listenTo = UserStore;
  }
  render() {
    return <div className="user">{this.state.user}</div>;
  }
  getState() {
    return {
      user: UserStore.getUser(this.props.id)
    }
  }
}

listenTo

Must be either a store or an array of stores. Just before the initial render, it will register a change listener with the specified store(s).

When the element is about to be unmounted from the DOM it will dispose of an change listeners.

class User extends Marty.Component {
  constructor(props, context) {
    super(props, context);
    this.listenTo = [UserStore, FriendsStore];
  }
  render() {
    return (
      <div className="user">
        <span className="name">{this.state.user.name}</span>
        <FriendsList friends={this.state.friends}/>
      </div>
    );
  }
  getState() {
    return {
      user: UserStore.getUser(this.props.id),
      friends: FriendsStore.getFriendsForUser(this.props.id)
    }
  }
}

getState

The result of getState will be set as the components state. getState will be called

  • When the component is being constructed
  • The components props change
  • When any of the stores specified by listenTo change
